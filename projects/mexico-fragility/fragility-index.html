Author

Published

This piece builds a state-level fragility index for Mexico from three dimensions, lethal violence, structural marginalization, and income poverty, and asks a spatial question of it: does fragility cluster, so that fragile states sit next to other fragile states? For Afghanistan the answer was yes, and the corridors of transmission were the story. For Mexico the answer is no, and the reason is the finding. Violence concentrates in the north and along the Pacific, deprivation concentrates in the south, and because the two run in opposite directions a single composite index cancels its own geography. Mexico does not have one fragility map. It has two, and they disagree.

NoteData access

Violence is from SESNSP (official reported crime) and ACLED (political and cartel violence). Marginalization is CONAPO’s normalized index, 2020. Poverty is INEGI’s 2024 multidimensional measurement, which INEGI now produces following the dissolution of CONEVAL. Boundaries are GADM v4.1. ACLED restricts redistribution and INEGI, CONAPO, and SESNSP are open government data; the raw inputs are not committed to this repository. See data/README.md for how to obtain them.

Question

Where is fragility highest across Mexico’s 32 states, and is it a spatial phenomenon? A fragility index that combines several dimensions is only useful if those dimensions describe the same places. If they describe different places, the composite is misleading and a typology is the honest tool. The spatial test, whether the index clusters, is what reveals which case Mexico is in.

Data and methods

Fragility is built from three standardized dimensions. Violence is the average of two z-scored rates per 100,000, intentional homicides from SESNSP and conflict fatalities from ACLED, both averaged over 2019 to 2024. Deprivation is CONAPO’s normalized marginalization index for 2020, which folds together education, housing, services, and income. Poverty is the share of the population in multidimensional poverty, derived from INEGI’s 2024 counts over state population. Each dimension is converted to a z-score and the composite is their mean, with a PCA-weighted version reported for comparison. Spatial structure is tested with Moran’s I on a queen-contiguity graph of the states, run on each dimension and on the composite.

Findings

Three dimensions, three different maps

The first finding is visible before any statistic. Violence is a northern and Pacific phenomenon. Deprivation and poverty are southern. The maps of the three standardized dimensions barely overlap.

Quantitatively, each dimension clusters on its own. Moran’s I is -0.014 for violence (p 0.44), 0.392 for marginalization (p 2.2^{-4}), and 0.523 for poverty (p 3.3^{-6}). Each is a coherent regional pattern. The problem is that they are different regions.

The correlation between violence and poverty across states is -0.13, negative. The violent states, Colima, Baja California, Sonora, Chihuahua, are relatively wealthy; the poorest states, Chiapas, Guerrero, Oaxaca, are comparatively less violent. Fragility in Mexico is not one condition with one cause. It is at least two.

The composite, and why it under-clusters

Averaging the three dimensions produces a defensible ranking, but it inherits the contradiction. Because violence pulls the index up in the north and poverty pulls it up in the south, the composite has almost no spatial signal of its own: Moran’s I is -0.012 with p 0.43, not significant, even though every component clusters. The composite map looks scattered not because fragility is random but because it is the sum of two opposite geographies.

Limitations

The unit is the state, and there are only 32, so the spatial tests are suggestive rather than strong, and the contiguity graph is coarse. The natural next iteration is the municipal level, where the same decoupling can be tested across roughly 2,475 units and the spatial structure has room to show itself.

The poverty rate is computed from INEGI’s 2024 poverty counts divided by 2020 CONAPO population, so the absolute percentages run slightly high, Chiapas computes near 70 against the official 66. The ranking the index uses is unaffected, since the overstatement is roughly proportional, but the displayed percentages should be read as approximate.

The dimensions carry different vintages: violence is 2019 to 2024, marginalization is the 2020 census, and poverty is 2024. The index treats them as one snapshot. SESNSP and ACLED both capture lethal violence, so combining them gives homicide somewhat more weight than a single source would; this is deliberate, to use both as requested, but it is a choice worth noting.

The composite is equal-weighted. The PCA-weighted version is computed alongside and tells the same story, because the decoupling is a property of the data, not of the weighting.

```{r}
state_key <- function(x) {
  x <- stri_trans_general(as.character(x), "Latin-ASCII"); x <- tolower(str_squish(x))
  recode(x,
    "distrito federal" = "ciudad de mexico", "cdmx" = "ciudad de mexico",
    "mexico city" = "ciudad de mexico",
    "coahuila de zaragoza" = "coahuila", "michoacan de ocampo" = "michoacan",
    "veracruz de ignacio de la llave" = "veracruz", "veracruz llave" = "veracruz",
    "queretaro de arteaga" = "queretaro", "estado de mexico" = "mexico")
}
pick_rx <- function(rx, cols) { h <- cols[str_detect(cols, rx)]; if (length(h)) h[1] else NA_character_ }
z <- function(x) (x - mean(x, na.rm = TRUE)) / sd(x, na.rm = TRUE)
read_table_find <- function(path, must_match) {
  is_csv <- grepl("\\.csv$", path, ignore.case = TRUE)
  for (sk in 0:12) {
    d <- tryCatch(
      if (is_csv) suppressMessages(read_csv(path, skip = sk, show_col_types = FALSE,
                                            locale = locale(encoding = "Latin1")))
      else suppressMessages(read_excel(path, skip = sk)), error = function(e) NULL)
    if (is.null(d) || ncol(d) < 3) next
    nms <- stri_trans_general(tolower(names(d)), "Latin-ASCII"); names(d) <- nms
    auto <- str_detect(nms, "^\\.\\.\\.\\d+$") | nms == "" | is.na(nms)
    if (any(str_detect(nms, must_match) & nchar(nms) <= 40 & !auto) && sum(!auto) >= 3) return(d)
  }
  stop("No header matching '", must_match, "' in ", path)
}
```

```{r}
mi <- function(v) moran.test(v, lw, zero.policy = TRUE)
m_viol <- mi(sx$viol_dim); m_depr <- mi(sx$depr_dim); m_pov <- mi(sx$pov_dim); m_comp <- mi(sx$fragility)
r_vp <- cor(df$viol_dim, df$pov_dim)
```

## Findings

### Three dimensions, three different maps

The first finding is visible before any statistic. Violence is a northern and Pacific phenomenon. Deprivation and poverty are southern. The maps of the three standardized dimensions barely overlap.

Quantitatively, each dimension clusters on its own. Moran's I is `r round(m_viol$estimate[1], 3)` for violence (p `r signif(m_viol$p.value, 2)`), `r round(m_depr$estimate[1], 3)` for marginalization (p `r signif(m_depr$p.value, 2)`), and `r round(m_pov$estimate[1], 3)` for poverty (p `r signif(m_pov$p.value, 2)`). Each is a coherent regional pattern. The problem is that they are different regions.

### Violence and poverty pull apart

The correlation between violence and poverty across states is `r round(r_vp, 2)`, negative. The violent states, Colima, Baja California, Sonora, Chihuahua, are relatively wealthy; the poorest states, Chiapas, Guerrero, Oaxaca, are comparatively less violent. Fragility in Mexico is not one condition with one cause. It is at least two.

### The composite, and why it under-clusters

Averaging the three dimensions produces a defensible ranking, but it inherits the contradiction. Because violence pulls the index up in the north and poverty pulls it up in the south, the composite has almost no spatial signal of its own: Moran's I is `r round(m_comp$estimate[1], 3)` with p `r signif(m_comp$p.value, 2)`, not significant, even though every component clusters. The composite map looks scattered not because fragility is random but because it is the sum of two opposite geographies.

## Limitations

The unit is the state, and there are only 32, so the spatial tests are suggestive rather than strong, and the contiguity graph is coarse. The natural next iteration is the municipal level, where the same decoupling can be tested across roughly 2,475 units and the spatial structure has room to show itself.

The poverty rate is computed from INEGI's 2024 poverty counts divided by 2020 CONAPO population, so the absolute percentages run slightly high, Chiapas computes near 70 against the official 66. The ranking the index uses is unaffected, since the overstatement is roughly proportional, but the displayed percentages should be read as approximate.

The dimensions carry different vintages: violence is 2019 to 2024, marginalization is the 2020 census, and poverty is 2024. The index treats them as one snapshot. SESNSP and ACLED both capture lethal violence, so combining them gives homicide somewhat more weight than a single source would; this is deliberate, to use both as requested, but it is a choice worth noting.

The composite is equal-weighted. The PCA-weighted version is computed alongside and tells the same story, because the decoupling is a property of the data, not of the weighting.
